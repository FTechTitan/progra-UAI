2 · Indexación + slicing

Positiva desde 0, negativa desde −1, slicing con dos puntos:

datos = [4, 8, 15, 16, 23]
datos[0]     # 4    (primero)
datos[-1]    # 23   (último)
datos[1:3]   # [8, 15]  (1 hasta 3 sin incluir)

3 · Métodos

Arsenal nativo para mutar la lista:

xs = [10, 20, 30]
xs.append(40)     # agrega al final
xs.insert(0, 5)   # inserta en índice 0
xs.remove(20)     # borra el valor 20
xs.pop()          # saca el último
xs[0] = 99        # sobrescribe por posición

4 · Producto punto

Recorrer dos listas en paralelo y acumular:

a = [2, 9, 3, 10, 10]
b = [3, 7, 5, 1, 6]
suma = 0
for i in range(5):
    suma += a[i] * b[i]
print("Producto punto:", suma)   # 154

1-NN: vecino más cercano

Tres listas paralelas (x, y, etiqueta) e índices:

import math
menor, mejor = float("inf"), 0
for i in range(len(xs)):
    d = math.dist((px, py), (xs[i], ys[i]))
    if d < menor:
        menor, mejor = d, i
print("Clase:", etiquetas[mejor])

Lero Mart: 3×2

Ordenar de mayor a menor y saltar de 3 en 3:

precios.sort(reverse=True)        # mayor a menor
total = 0
for i in range(0, len(precios), 3):
    total += precios[i]           # más caro
    if i + 1 < len(precios):
        total += precios[i + 1]   # 2º (el 3º gratis)
