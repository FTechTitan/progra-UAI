2 · while — el guardián

Evalúa la condición ANTES de cada vuelta:

contador = 0
while contador < 5:
    print(contador)
    contador += 1     # ← actualizar la condición

while + continue

Repetir hasta que el usuario teclee 0:

n = int(input("Número (0 corta): "))
while n != 0:
    if n > 0:
        print("Positivo")
    n = int(input("Otro: "))   # ← volver a preguntar

4 · for vs while

Mismo resultado, el for es más compacto:

# while → 4 líneas
i = 0
while i < 5:
    print("#", end=""); i += 1

# for → 2 líneas (incremento automático)
for i in range(5):
    print("#", end="")

5 · Cajero (contador)

Máximo 3 intentos: patrón contador dentro de while:

intentos = 0
clave = "1234"
pin = input("PIN: ")
while intentos < 3 and pin != clave:
    intentos += 1
    pin = input("PIN incorrecto, reintentá: ")

Acumulador

Suma valores variables hasta una señal de parada (−1):

suma = 0
n = int(input("Número (-1 termina): "))
while n != -1:
    suma += n          # acumula
    n = int(input(": "))
print("Suma:", suma)

Robot: máquina de estados

Procesa comandos y actualiza coordenadas hasta 'salir':

import math
x = y = 0
orden = input("Comando: ")
while orden != "salir":
    if orden == "adelante": y += 1
    elif orden == "derecha":  x += 1
    orden = input("Comando: ")
print("Distancia:", math.sqrt(x**2 + y**2))
