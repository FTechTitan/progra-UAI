if · elif · else

Encadená varias decisiones sin romper el flujo:

if nota >= 4.0:
    print("Aprobado")
elif nota >= 3.0:
    print("En riesgo")
else:
    print("Reprobado")

3 · Secuencial vs anidado

Una sola pregunta, dos caminos excluyentes (operador módulo %):

n = int(input("Número: "))
if n % 3 == 0:
    print("Múltiplo de 3")
else:
    print("No es múltiplo")

Condicionales anidados

    if corte == "vacuno":

4 · Sensor IoT (°F → °C)

Capturar → convertir → clasificar con if/elif/else:

f = float(input("Temp °F: "))
c = (f - 32) / 1.8
print("Celsius:", c)

if c < 15:
    print("Hace frío")
elif c <= 25:
    print("Temperatura agradable")
else:
    print("Hace calor")

5 · Empaquetado (// y %)

División entera y resto deciden si hace falta una caja extra:

cubos = 80
por_caja = 16
llenas = cubos // por_caja   # división entera → 5
sobran = cubos % por_caja    # resto → 0

if sobran != 0:
    print(llenas + 1, "cajas (una parcial)")
else:
    print(llenas, "cajas exactas")
